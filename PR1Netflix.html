Analysis of impact of Netflix's feedback system on its User Experience

Coursework: IMT 570 - Analytical Mehods for Information Professionals | October 2018 - December 2018

See Project Report

The goal of the group research project was to learn about the research process by completing a small-scale study. The research project encompassed planning, data collection, analysis and making recommendations. Key components were :

  1. Defining Research Problem
  2. Research Design and Methodology
  3. Qualitative and Quantitative Analysis

Steps and Outcomes:

  • Analyzed the impact of recent changes in Netflix’s long-standing feedback system on its overall user experience.
  • Examined the existing literature. Conducted surveys using cluster and purposive sampling, and interviews using convenient sampling.
  • The survey concluded that close to 75% participants mentioned that they didn’t notice the removal of user reviews from Netflix. However, when presented with the screenshots of the Old and New implementations of Netflix’s feedback system, close to 70% of the users preferred the old implementation with ‘5-star’ rating system and user reviews
  • This result aligned with findings from the interviews where participants mentioned that having the older implementation of the feedback system can significantly improve their user experience on Netflix
  • Concluded that the recent changes to the feedback system by Netflix have not directly impacted the user experience
  • However, the following are the recommendations from the study:
    1. Netflix can better its user experience by bringing back user reviews and the ‘5-star’ rating system.
    2. Netflix can launch a new campaign to better market this sought after feature.
    3. The streaming service provider can invest in bringing consistency in its feedback system across all its platforms like the Website, Web Application, and Mobile Application.

Key Learnings

Planning research. Leveraging existing literature and data for research, data collection methods using Interviews, focus groups, oservations and surveys. Exploring and analyzing data. Hypothesis Testing and Experimentation.
